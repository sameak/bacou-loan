## How to rebuild:

1. Go to github.com/sameak/bacou-loan
2. Click **Actions** tab
3. Click **Production Build** workflow
4. Click **Run workflow** → select platform → click **Run**

Or run manually:
```
eas build --profile production --platform ios
```
